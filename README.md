# Patient Repository Readme
## **Prerequisites:**

Before downloading the github repository locally, make sure you have the following installed:

* Java 21 (JDK 21)

* A Java-compatible IDE (IntelliJ IDEA, Eclipse, etc.)

* Windows OS (recommended for Bluetooth compatibility)


#### **Dependencies:**

* io.ultreia:bluecove:2.1.1 — Bluetooth communication

* org.apache.commons:commons-lang3:3.15.0 — Apache utility library


#### **Hardware:**

* A BITalino device


## **Installation:**

  1. Clone the repository locally

  2. Build the project by: mvn clean install

  3. Run the _UI_ class
